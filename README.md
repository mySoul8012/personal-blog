# personal-blog
个人Blog开源项目

该项目是一个 Express.js 框架为基础的，MongoDB 作为数据库的，前后端分离，前端为 Vue.js 的个人小 Blog 项目。其中前后端的通信使用的是基本的 Restful API 实现。安全校验使用 JWT 完成。

具体的功能有安全验证，用户访问前端的时候，可以查看最新博文，以及博文详情，对博文进行评论，管理员登录后台以后，可以对网站的文章进行管理，可以对评论进行管理。
